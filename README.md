## إضافة أسئلة جديدة

### تركيبة الفولدرات
```
questions/
  مسلسلات/
    Breaking_Bad.json
    Game_of_Thrones.json
  تاريخ/
    Islamic_History.json
    Egypt_History.json
  رياضة/
    كرة_قدم.json
  علوم/
    فيزياء.json
```

### شكل ملف JSON
```json
{
  "category": "اسم الكاتيجوري",
  "questions": [
    { "points": 200, "q": "نص السؤال", "a": "الإجابة" },
    { "points": 200, "q": "نص السؤال", "a": "الإجابة" },
    { "points": 400, "q": "نص السؤال", "a": "الإجابة" },
    { "points": 400, "q": "نص السؤال", "a": "الإجابة" },
    { "points": 600, "q": "نص السؤال", "a": "الإجابة" },
    { "points": 600, "q": "نص السؤال", "a": "الإجابة" }
  ]
}
```

**ملاحظة:** كل لعبة بتستخدم 2 سؤال ×200 + 2×400 + 2×600
لو عندك 12 سؤال = 2 لعبة ممكنة، 18 سؤال = 3 لعبة، إلخ

## الأسئلة المستخدمة

بتتحفظ في: `data/used_questions.json`
ده ملف حقيقي على جهازك، مش مؤقت.

لو عايز تعيد أسئلة كاتيجوري معينة:
- امسح إدخالها من ملف `data/used_questions.json`
- أو من واجهة الموقع (قريباً)
